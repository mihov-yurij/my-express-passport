# Express Modular API (MVC)

Учебный проект модульного сервера на Node.js и Express с использованием паттерна MVC.

## Функционал

- Реализован CRUD для пользователей и статей.
- Использование паттерна MVC (Routes, Controllers, Middlewares).
- Централизованная обработка ошибок.
- Валидация числовых ID через Middleware.
- Логирование запросов с помощью Morgan.
- Постоянное хранение данных в JSON-файлах.
