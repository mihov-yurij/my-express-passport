# Express Cookies & Template Engine (MVC)

Учебный проект на Node.js и Express, посвященный работе с **Cookies**, управлению состоянием пользователя и гибридному использованию шаблонизаторов в архитектуре MVC.

## 🌟 Ключевые особенности

- **Работа с Cookies**: Реализовано чтение и установка кук для сохранения предпочтений пользователя (например, темы оформления или данных сессии).
- **Двойной рендеринг (Hybrid Views)**:
  - **EJS**: Используется для контента статей (`articles.ejs`, `article-details.ejs`).
  - **PUG**: Используется для работы с пользователями и общей разметки (`user.pug`, `layout.pug`).
- **MVC Структура**: Логика приложения разделена на контроллеры, модели (через `utils`) и представления.
- **Утилиты**: Папка `src/utils` содержит хелперы для асинхронного чтения/записи данных в JSON-файлы.

## 📂 Структура папок

- `src/utils/` — Логика работы с данными (чтение/запись JSON).
- `src/views/` — Шаблоны страниц (.ejs и .pug).
- `app.js` — Конфигурация Express, подключение `cookie-parser` и статики.
- `server.js` — Точка входа, запуск сервера на порту.
