# 🚀 Express Secure App: Cookies, JWT & Hybrid MVC

Учебный проект на **Node.js** и **Express**, демонстрирующий продвинутую работу с безопасностью, состоянием пользователя через **Cookies/JWT** и гибкую настройку шаблонизаторов в архитектуре **MVC**.

## 🌟 Ключевые возможности

- **🛡 Безопасность с Helmet**: Настроены HTTP-заголовки и строгая политика **Content Security Policy (CSP)** для защиты от XSS-атак.
- **🔑 Аутентификация через JWT**: 
  - Генерация **JSON Web Tokens** при входе.
  - Безопасное хранение токенов в куках с флагом `httpOnly`.
- **🎨 Кастомизация (Theme Switcher)**: Реализовано сохранение темы оформления (Light/Dark) в браузере пользователя через `cookie-parser`.
- **🎭 Гибридный рендеринг**:
  - **EJS**: Используется для страниц статей и динамических списков.
  - **Pug**: Используется для системных интерфейсов и общих макетов.
- **📂 MVC Структура**: Логика разделена на маршруты (routes), представления (views) и вспомогательные утилиты (utils).

## 🛠 Технологический стек

- **Backend**: Node.js, Express.js
- **Безопасность**: Helmet, JWT (jsonwebtoken)
- **Парсинг**: cookie-parser, body-parser
- **Логирование**: Morgan
- **Шаблонизаторы**: EJS, Pug

## 📂 Структура проекта

```text
├── public/              # Статические файлы (CSS, JS, изображения)
