# Express Modular API & Template Engines (MVC)

Учебный проект модульного сервера на Node.js и Express с использованием паттерна MVC и поддержкой нескольких шаблонизаторов.

## Функционал

- **Многоканальный рендеринг**: Использование **PUG** для работы с пользователями и **EJS** для работы со статьями.
- **CRUD операции**: Реализовано управление пользователями и статьями.
- **Архитектура MVC**: Четкое разделение на Роуты, Контроллеры и Представления (Views).
- **Middleware**: Валидация числовых ID и централизованная обработка ошибок.
- **Статика**: Подключены CSS-стили для оформления страниц.
- **Хранение данных**: Данные сохраняются в JSON-файлах через файловый хелпер.
